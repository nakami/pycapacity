pycapacity.visual moduleÔÉÅ

OverviewÔÉÅ

This is a pyhton module helping to visualise 2d and 3d polytopes and ellipsoids. It is based on the module matplotlib.

  • visualising 2d and 3d polytope faces and vertices

  • visualising 2d and 3d ellipsoids

pycapacity.visual.plot_ellipsoid(radii, rotation, center=None, ax=None, plt=None, label=None, color=None, edge_color=None, alpha=1.0)[source]ÔÉÅ

Plotting ellipsoid in 2d and 3d

Note

If provided matplotlib plt it will find the ax automatically. No need to provide the ax. If ax already defined no need to provide the plt.

Examples

>>> import pycapacity.visual
>>> import matplotlib.pyplot as plt
>>> import numpy as np
>>> ax = pycapacity.visual.plot_ellipsoid([6,5,4],np.eye(3),plt=plt, color='blue', alpha=0.5)
>>> pycapacity.visual.plot_ellipsoid([1,2,3],np.eye(3),ax=ax, color='red', alpha=0.5)
>>> plt.show()
Parameters:

  • radii ‚Äì radii of the ellipsoid in each axis

  • rotation ‚Äì rotation matrix

  • center ‚Äì offset of the ellispoid from origin

  • ax ‚Äì matplotlib ax to plot on

  • plt ‚Äì matplotlib plot to plot on - it will find the ax automatically

  • color ‚Äì face color

  • edge_color ‚Äì egde collor

  • alpha ‚Äì opacity

  • label ‚Äì legend label

Returns:

matplotlib ax used for plotting

Return type:

ax
