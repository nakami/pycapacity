pycapacity.human.acceleration_polytope(J, N, M, F_min, F_max, tol=1e-05)[source]ÔÉÅ

A function calculating the polytopes of achievable accelerations based on the jacobian matrix J, moment arm matrix N and mass matrix M

\[P_{a} = \{ \ddot{x} ~ | ~ \ddot{x} = JM^{-1}NF, \quad F_{min} \leq F \leq F_{max}\}\]

Based on the hyper_plane_shifting_method algorihtm.

Parameters:

  • J ‚Äì jacobian matrix

  • N ‚Äì moment arm matrix

  • M ‚Äì mass matrix

  • F_min ‚Äì minimal muscular forces (passive forces or 0)

  • F_max ‚Äì maximal isometric forces

  • tolerance ‚Äì tolerance for the polytope calculation

Returns:

  • a_vert(array) ‚Äì list of acceleraiton vertices

  • H(array) ‚Äì half-space rep matrix H - \(Ha < d\)

  • d(array) ‚Äì half-space rep vectors d

  • faces ‚Äì indexes of verteices forming the polytope faces

pycapacity.human.force_polytope(J, N, F_min, F_max, tol, torque_bias=None)[source]ÔÉÅ

A function calculating the polytopes of achievable foreces based on the jacobian matrix J and moment arm matrix N

\[P_{f} = \{ f ~ | ~ J^Tf = NF, \quad F_{min} \leq F \leq F_{max}\}\]

optionally an additional bias \(t_{bias}\) can be added

\[P_{f} = \{ f ~ | ~ J^Tf = NF + t_{bias}, \quad F_{min} \leq F \leq F_{max}\}\]

Based on the iterative_convex_hull_method algorihtm.

Parameters:

  • J ‚Äì jacobian matrix

  • N ‚Äì moment arm matrix

  • F_min ‚Äì minimal muscular forces (passive forces or 0)

  • F_max ‚Äì maximal isometric forces

  • tolerance ‚Äì tolerance for the polytope calculation

  • torque_bias ‚Äì torque bias optional (gravity or movement or applied forces ‚Ä¶.)

Returns:

  • f_vert(list) ‚Äì list of cartesian force vertices

  • H(array) ‚Äì half-space rep matrix H - \(Hf < d\)

  • d(array) ‚Äì half-space rep vectors d

  • faces(list) ‚Äì list of vertex indexes forming polytope faces

A function calculating the polytopes of achievable velocity based on the jacobian matrix J and moment arm matrix N

\[P_{v} = \{ \dot{x} ~ | ~ L\dot{q} = \dot{l},~~ J\dot{q} = \dot{x}, \quad \dot{l}_{min} \leq \dot{l} \leq \dot{l}_{max}\}\]

Based on the iterative_convex_hull_method algorihtm.

Parameters:

  • J ‚Äì jacobian matrix

  • N ‚Äì moment arm matrix \(L = -N^T\)

  • dl_min ‚Äì minimal achievable muscle contraction veclocity

  • dl_max ‚Äì maximal achievable muscle contraction veclocity

  • tol ‚Äì tolerance for the polytope calculation

Returns:

  • v_vert(list) ‚Äì list of cartesian velocity vertices

  • H(array) ‚Äì half-space rep matrix H - \(Hv < d\)

  • d(array) ‚Äì half-space rep vectors d

  • faces(list) ‚Äì list of vertex indexes forming velocity polytope faces
